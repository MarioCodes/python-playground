All this tools are built through Poetry. [How to use it](https://mariocod.es/python/poetry.html)

## AI Learning
Code and mini projects I did to learn AI programming following the book [Prompt Engineering](https://www.oreilly.com/library/view/prompt-engineering-for/9781098153427/) and Medium.  
It includes:
* [Call to open AI](https://github.com/MarioCodes/python-playground/blob/master/AiLearning/Testing/CallToOpenAI.py) to query ChatGPT's API
* LangChain usage [mini example](https://github.com/MarioCodes/python-playground/tree/master/AiLearning/LangChain)
* [Here](https://github.com/MarioCodes/python-playground/tree/master/AiLearning/InsertAndQueryVectorDatabase) how to insert and how to query vector databases (CosmosDB) using an embedding model and doing a vectorial search.
* Also how to use a [RAG system](https://github.com/MarioCodes/python-playground/tree/master/AiLearning/Rag) to include context to your queries from propietary files. It includes reading, chunking, inserting and then (in another class) querying.
