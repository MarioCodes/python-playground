All this tools are built through Poetry. [Quick guide on how to use it](https://mariocod.es/python/poetry.html)

## Prompt engineering
Code and mini projects I did to learn AI programming following the book [Prompt Engineering](https://www.oreilly.com/library/view/prompt-engineering-for/9781098153427/) and Medium.  
It includes:
* [Call to open AI](https://github.com/MarioCodes/python-playground/blob/master/PromptEngineering/CallToOpenAI.py) to query ChatGPT's API
* LangChain usage [mini example](https://github.com/MarioCodes/python-playground/tree/master/PromptEngineering/LangChain)
* [Here](https://github.com/MarioCodes/python-playground/tree/master/PromptEngineering/Embeddings) how to transform text into vectors using an embeddings model and insert it into CosmosDB. Then how to query another text by proximity using vectors against CosmosDB.
